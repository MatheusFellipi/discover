# HTML5 Semântico

- As tags elas quer irão ajudar a criar um HTML semântico,

* Secções comuns em documentos HTML
  - cabeçalhos `<header>`
  - navegação `<nav>`
  - barra lateral `<aside>`
  - rodapé `<footer>`

## Header

```html
<body>
  <header></header>

  <section>
    <header></header>
  </section>
</body>
```
